# Вступительная
Данный порт написан на языке C#, с использованием технологии дизайна MAUI.
Прототип находится на сыром уровне и нужен для демонстрации возможности адаптации WPF-подобного дизайна под экран мобильного телефона.

## Описание
Мобильная версия игры 2048 с вертикальной ориентацией. Игра содержит 4 столбца для размещения карточек с числами, которые можно объединять при совпадении значений.

## Функциональность
- 4 игровых столбца
- Выбор столбца для размещения карточки
- Автоматическое объединение одинаковых значений
- Цветовая индикация карточек разного номинала
- Подсчет очков

## Требования
- .NET 8.0
- Visual Studio 2022 или Visual Studio Code
- .NET MAUI Workload
- Android SDK (для Android сборки)
